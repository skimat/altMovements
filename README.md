altMovements
============
This keyboard shortcuts will enable you to make any movmements in a text edtor much easier.
All the common operations like up , down , left , right movements , selections and erase, cut-copy-paste

^!+r::reload
!i::send, {up}
!j::send, {left}
!k::send, {down}
!l::send, {right}
!m::send, {end}
!n::send, {backspace}
!p::send, {delete}
!h::send, {home}
!x::send, {x}
!c::send, {c}
!v::send, {v}


!^+i::send, ^+{up}
!^+j::send, ^+{left}
!^+k::send, ^+{down}
!^+l::send, ^+{right}



!^i::send, ^{up}
!^j::send, ^{left}
!^k::send, ^{down}
!^l::send, ^{right}

!+i::send, +{up}
!+j::send, +{left}
!+k::send, +{down}
!+l::send, +{right}
!+m::send, +{end}
!+h::send, +{home}
